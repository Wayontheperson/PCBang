# 개관


# 기능
* PC방 관리 프로그램

### 고객
* 자리 내 음식 주문 및 시간 연장
* 실시간으로 관리자와 채팅 가능
* 실시간 게임 순위 확인 (게임 타이틀 더블클릭 수에 따라서 순위가 바뀝니다.)
* 사용 종료시 사용자의 잔여 시간 자동 저장

### 관리자
* 실시간으로 고객과 채팅 가능
* 고객 사용시간 관리 가능
* 사용중인 고객 확인
* 매출 확인
* 회원 관리




# 사용기술
### 언어
* C# 3.0+

### 프레임워크
* .NET Framework 4.8
* Winform
* Entity Framework

### 서드파티 컨트롤
* DevExpress WinForms

### 데이터베이스
* MS-SQL Server 2019
