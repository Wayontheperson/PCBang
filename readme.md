# 개관


# 기능
* PC방 관리 프로그램

### 고객
* 빠른 게임 접속
* 음식 주문 및 시간 연장
* 실시간으로 관리자와 채팅
* 실시간 게임 순위 확인 (클릭 수에 따라서 순위가 바뀝니다.)

### 관리자
* 실시간으로 고객과 채팅
* 고객 시간 추가
* 사용중인 고객 확인
* 매출 확인
* 회원 관리




# 사용기술
### 언어
* C# 3.0+

### 프레임워크
* .NET Framework 4.8
* Winform
* Entity Framework

### 서드파티 컨트롤
* DevExpress

### 데이터베이스
* MS-SQL Server 2019

# 미해결 부분
* 월간 게임순위
* 환불 계산
